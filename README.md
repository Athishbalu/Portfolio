# Portfolio
# 🚀 Athi Developer Portfolio

A modern, responsive, and animated developer portfolio built using **HTML, CSS, and JavaScript**.
Designed to showcase projects, skills, and contact details with a clean UI and smooth user experience.

---

## ✨ Features

* 🌗 Dark / Light theme toggle
* ⌨️ Typing animation effect
* 🎯 Scroll-based reveal animations
* 📊 Animated stats counter
* 💻 Responsive design (Mobile + Desktop)
* 🧠 Skills section with progress bars
* 🔗 GitHub live project integration
* 📩 Contact form (frontend simulation)
* 🖱️ Custom cursor interaction

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Custom properties + animations)
* JavaScript (Vanilla JS)
* GitHub API

## ⚙️ How to Run

1. Download or clone the project
2. Open the folder
3. Double-click `index.html`

OR

Use Live Server in VS Code:

```
Right click → Open with Live Server
```

## 🔧 Customization

### Change Projects

Edit inside `index.html` → Projects section

### Update GitHub Username

In `script.js`:

```js
const GITHUB_USERNAME = 'your-username';
```

### Change Stats (Projects / Clients / Years)

Update:

```html
<h3 class="hcn-n" data-t="4">0</h3>
```
